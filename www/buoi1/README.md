# Vật Liệu Xây Dựng - Demo

Hướng dẫn nhanh:

- Đặt ảnh logo (ảnh số 2 từ phần đính kèm) vào `images/logo.png` trong thư mục `c:/laragon/www/buoi1/images/logo.png`.
- Mở trình duyệt và truy cập `http://localhost/buoi1/` (hoặc `http://127.0.0.1/buoi1/`) nếu Laragon/Apache đang chạy.

Ghi chú:

- Tệp `index.php` có giao diện mẫu (header, search, menu, banner) và liên kết tới `assets/style.css`.
- Nếu muốn thay đổi logo, ghi đè file `images/logo.png` bằng ảnh bạn muốn.
