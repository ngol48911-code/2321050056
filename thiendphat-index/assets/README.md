Hướng dẫn logo

1. Bạn có kèm file logo (attachment) trong cuộc trò chuyện — lưu file đó dưới tên `logo.png`.
2. Tạo thư mục `assets` (nếu chưa có) và đặt `logo.png` vào `d:/2321050056/thiendphat-index/assets/logo.png`.
3. `index.html` đã tham chiếu `assets/logo.png` — mở `index.html` trong trình duyệt để xem logo.

Ghi chú: nếu cần đổi tên file (ví dụ `logo.jpg`), cập nhật đường dẫn trong `index.html`.
